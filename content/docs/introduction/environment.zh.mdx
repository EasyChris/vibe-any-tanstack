## 环境变量系统介绍
VibeAny 配置环境变量有两种方式：
1. 通过环境变量文件配置，对于一些不常变化的环境变量，可以配置在环境变量文件中，比如数据库连接URL、鉴权密钥等。
2. 通过后台配置面板配置，对于一些常变化的环境变量，VibeAny 为你提供动态配置面板，你可以在后台配置面板中随时修改环境变量，比如支付密钥、邮件服务器配置等。

## 通过环境变量文件配置
### 数据库
在数据库设置文档中查看更加详细的配置说明
| 变量   | 描述 |
| -------- | ------- |
| DATABASE_URL  | 数据库连接 URL    |

### 鉴权
在鉴权设置文档中查看更加详细的配置说明
| 变量   | 描述 |
| -------- | ------- |
| ADMIN_EMAILS  |  管理员邮箱    |
| BETTER_AUTH_SECRET  |  鉴权密钥，可以在[better-auth](https://www.better-auth.com/docs/installation#set-environment-variables)官网生成    |
| BETTER_AUTH_URL  |  鉴权URL    |

如果你想使用OAuth（支持github或者谷歌登入），可以继续配置如下内容：
| 变量   | 描述 |
| -------- | ------- |
| GITHUB_CLIENT_ID  |  GitHub OAuth 客户端 ID    |
| GITHUB_CLIENT_SECRET  |  GitHub OAuth 客户端密钥    |
| GOOGLE_CLIENT_ID  |  Google OAuth 客户端 ID    |
| GOOGLE_CLIENT_SECRET  |  Google OAuth 客户端密钥    |

### 邮件
你可以使用自己的邮件服务器，也可以使用 resend 服务。
| 变量   | 描述 |
| -------- | ------- |
| EMAIL_HOST  |  邮件服务器主机    |
| EMAIL_ADDRESS  |  发件人邮箱    |
| EMAIL_USER  |  发件人用户名    |
| EMAIL_PASSWORD  |  发件人密码    |

二者择一即可，如果同时配置，默认使用 resend 服务。
| 变量   | 描述 |
| -------- | ------- |
| RESEND_API_KEY  |  resend 服务 API 密钥    |
### 支付 / 积分

#### Stripe 服务
| 变量 | 描述 |
| -------- | ------- |
| NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY | Stripe 服务 publishable 密钥 |
| STRIPE_SECRET_KEY | Stripe 服务 API 密钥 |
| STRIPE_WEBHOOK_SECRET | Stripe 服务 webhook 密钥 |
| NEXT_PUBLIC_STRIPE_PRO_MONTHLY_PRICE_ID | Stripe 服务 pro 每月价格 ID |
| NEXT_PUBLIC_STRIPE_PRO_YEARLY_PRICE_ID | Stripe 服务 pro 每年价格 ID |
| NEXT_PUBLIC_STRIPE_LIFETIME_PRICE_ID | Stripe 服务 lifetime 价格 ID |
| NEXT_PUBLIC_STRIPE_100_CREDITS_PRICE_ID | Stripe 服务 100 积分价格 ID(如果不需要积分功能，可以不配置) |
| NEXT_PUBLIC_STRIPE_200_CREDITS_PRICE_ID | Stripe 服务 200 积分价格 ID(如果不需要积分功能，可以不配置) |

#### Creem 服务
| 变量 | 描述 |
| -------- | ------- |
| CREEM_X_API_KEY | Creem 服务 API 密钥 |
| CREEM_TEST_MODE | Creem 服务测试模式 |
| CREEM_WEBHOOK_SECRET | Creem 服务 webhook 密钥 |
| NEXT_PUBLIC_CREEM_PRO_MONTHLY_PRODUCT_ID | Creem 服务 pro 每月产品 ID |
| NEXT_PUBLIC_CREEM_LIFETIME_PRODUCT_ID | Creem 服务 lifetime 产品 ID |
| NEXT_PUBLIC_CREEM_100_CREDITS_PRODUCT_ID | Creem 服务 100 积分产品 ID(如果不需要积分功能，可以不配置) |
| NEXT_PUBLIC_CREEM_200_CREDITS_PRODUCT_ID | Creem 服务 200 积分产品 ID(如果不需要积分功能，可以不配置) |
