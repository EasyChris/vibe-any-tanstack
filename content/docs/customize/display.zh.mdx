## 落地页配置

落地页的所有内容都在 `src/config/locale/landing.content.ts` 中配置，支持多语言。

### Banner 横幅

```ts
banner: {
  show: true,
  text: t({
    en: "Open Sourced in 2026: Production-grade AI SaaS template available!",
    zh: "2026 年开源：生产级 AI SaaS 模板现已发布！",
  }),
  button: {
    text: t({ en: "View on GitHub", zh: "在 GitHub 上查看" }),
    href: github,
  },
},
```

| 配置项 | 说明 |
| --- | --- |
| `show` | 是否显示 Banner |
| `text` | Banner 文本内容 |
| `button.text` | 按钮文本 |
| `button.href` | 按钮链接 |

## 定价页配置

定价页内容在 `src/config/locale/pricing.content.ts` 中配置。

套餐价格和订阅逻辑在 `src/config/website-config.ts` 中配置：

```ts title="src/config/website-config.ts"
export const websiteConfig: WebsiteConfig = {
  plans: [
    {
      id: "free",
      planType: "free",
      prices: [],
    },
    {
      id: "pro",
      planType: "subscription",
      credit: { amount: 100, expireDays: 31 },
      prices: [
        {
          priceId: import.meta.env.VITE_STRIPE_PRO_MONTHLY_PRICE_ID!,
          amount: 990, // $9.90
          currency: "USD",
          interval: "month",
        },
        // ...
      ],
      display: { isRecommended: true, group: "subscription" },
    },
  ],
}
```

<Callout type="warn" title="注意">
`priceId` 必须与支付服务商（如 Stripe）中创建的价格 ID 一致。
</Callout>
